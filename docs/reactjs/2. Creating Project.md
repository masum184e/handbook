## Create React App
Create React App (CRA) is an officially supported tool that sets up everything automatically.

### What CRA Provides

- Webpack & Babel preconfigured
- Development server
- Hot reloading
- Production build setup
- Zero configuration needed

### Steps
1. Create a React App
```sh
npx create-react-app my-react-app
```
- `npx` → Runs packages without installing globally
- `create-react-app` → React project generator
- `my-react-app` → Project name

2. Navigate to project folder `cd my-react-app`
3. Start Development Server `npm start`
    - App runs at: `http://localhost:3000`
    - Browser updates automatically on changes

### When to Use CRA
- ✅ Beginners
- ✅ Learning React fundamentals
- ❌ Slower builds
- ❌ Not ideal for large apps

## Vite
Vite is a modern build tool that is much faster than CRA.
### Steps
1. Create Vite + React App
```sh
npm create vite@latest my-vite-app
```
Then select:

- Framework → React
- Variant → JavaScript or TypeScript

2. Navigate to project folder `cd my-vite-app`
3. Install Dependencies `npm install`
4. Start Development Server `npm run dev`
    - App runs at: `http://localhost:3000`

### Why Vite is Better than CRA?

- Faster startup
- Faster builds
- Smaller bundle size
- Modern tooling

### When to Use Vite?

- Intermediate learners
- Production-ready projects
