Class Components

A class component is an ES6 class that:

  • Extends React.Component
  • Must have a render() method
  • Manages state using this.state

Syntax

import React from "react";

class Welcome extends React.Component {
  render() {
    return <h1>Hello, React!</h1>;
  }
}
  • extends React.Component gives React features
  • render() returns JSX
  • JSX defines what appears on the screen

Class Component with State

class Counter extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      count: 0,
    };
  }

  increment = () => {
    this.setState({ count: this.state.count + 1 });
  };

  render() {
    return (
      <div>
        <p>Count: {this.state.count}</p>
        <button onClick={this.increment}>Increment</button>
      </div>
    );
  }
}
  1. State is initialized in the constructor
  2. this.state.count accesses state
  3. this.setState() updates state
  4. Updating state triggers re-render

Event Handling in Class Components

Without arrow functions, you must bind this.

constructor(props) {
  super(props);
  this.handleClick = this.handleClick.bind(this);
}

Arrow functions automatically bind this.
