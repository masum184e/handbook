In **C#**, data structures are used to organize and store data efficiently. Here are some of the **most commonly used data structures** you’ll see in everyday C# development.

## Array

An **Array** stores elements of the **same type** in a fixed-size sequential collection.

- Fixed length
- Fast access by index
- Stored in contiguous memory

When an array is declared using `new`, its elements are automatically initialized with **default values**.  
For example, an `int` array will have the default value `0`.

```csharp
int[] numbers = new int[4];          // {0, 0, 0, 0}
int[] numbers = new int[4] { 10, 20, 30, 40 };
int[,] grid = new int[5, 8];         // 2D array
int[,,] building = new int[3, 4, 2]; // 3D array
```

**Basic Example**

```csharp
int[] numbers = { 10, 20, 30, 40 };

Console.WriteLine(numbers[0]); // Output: 10

foreach (int num in numbers)
{
    Console.WriteLine(num);
}
```

**Useful Properties**

- `.Length` → Returns the **total number of elements** in the array.
- `.GetLength(dimension)` → Returns the **size of a specific dimension** in a multi-dimensional array.

Example:

```csharp
int[,] matrix =
{
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};

foreach (int num in matrix)
{
    Console.Write(num + " ");
}
```

When iterating with `foreach`, the multi-dimensional array is **flattened** (elements are accessed sequentially).

**Use Cases**

- Storing a known number of items
- Fast index-based access

## Jagged Arrays

A **Jagged Array** is an **array of arrays**, where each inner array can have a different length.

```csharp
int[][] jaggedArray = new int[3][];

jaggedArray[0] = new int[] { 1, 2, 3 };
jaggedArray[1] = new int[] { 4, 5, 6 };
jaggedArray[2] = new int[] { 7, 8, 9 };

foreach (int[] row in jaggedArray)
{
    foreach (int num in row)
    {
        Console.Write(num + " ");
    }
    Console.WriteLine();
}
```

## List

`List<T>` is a **dynamic array** that can grow or shrink.

**Key points**

- Part of `System.Collections.Generic`
- Automatically resizes
- Easy insertion and removal

```csharp
List<string> names = new List<string>();
names.Add("Alice");
names.Add("Bob");
```

**Use cases**

- When the number of elements can change
- Most commonly used collection in C#

## Dictionary

`Dictionary<TKey, TValue>` stores **key-value pairs**.

**Key points**

- Fast lookup by key
- Keys must be unique
- Internally implemented using a hash table

```csharp
Dictionary<int, string> students = new Dictionary<int, string>();
students.Add(1, "John");
students.Add(2, "Sara");

Console.WriteLine(students[1]);
```

- `containsKey()` check wheather the key exist or not.

**Use cases**

- Fast searching
- Mapping relationships (ID → Name)

## Stack

`Stack<T>` follows **LIFO (Last In, First Out)**.

**Key points**

- Push → add item
- Pop → remove last item
- Peek → see top item

```csharp
Stack<int> stack = new Stack<int>();
stack.Push(1);
stack.Push(2);
stack.Pop(); // removes 2
```

**Use cases**

- Undo/redo operations
- Expression evaluation
- Function calls

## Queue

`Queue<T>` follows **FIFO (First In, First Out)**.

**Key points**

- Enqueue → add item
- Dequeue → remove first item

```csharp
Queue<string> queue = new Queue<string>();
queue.Enqueue("Task1");
queue.Enqueue("Task2");

queue.Dequeue(); // removes Task1
```

**Use cases**

- Task scheduling
- Request processing
- Messaging systems

## LinkedList

`LinkedList<T>` stores elements as **nodes connected by pointers**.

**Key points**

- Dynamic size
- Fast insertion/deletion
- No index-based access

```csharp
LinkedList<int> list = new LinkedList<int>();
list.AddLast(10);
list.AddLast(20);
```

**Use cases**

- Frequent insertions/deletions
- Implementing queues or stacks

## Quick summary

| Data Structure | Best For               |
| -------------- | ---------------------- |
| Array          | Fixed-size data        |
| List           | Dynamic collections    |
| Dictionary     | Fast key-value lookup  |
| Stack          | LIFO operations        |
| Queue          | FIFO operations        |
| LinkedList     | Frequent insert/delete |
