4. [Implement Queue using Stack](https://leetcode.com/problems/implement-queue-using-stacks/)
   1. 10 April, 2026: 00.11.25 ✔
        - Approach 1
            - Push everything into input
            - When you need to dequeue:
               - Move everything from input → output (reverses order) if output is empty
               - Then pop from output
               - Same for peak element
               - Empty check for both stack
        - Approach 2
            - Move all elements from input → output
            - Push element into input
            - Move everything back to input

            | Operation | **Approach 1: Lazy Transfer (Amortized)** | **Approach 2: Costly Push** |
            | --------- | ----------------------------------------- | --------------------------- |
            | Push      | O(1)                                      | O(n)                        |
            | Pop       | O(1) amortized / O(n) worst               | O(1)                        |
            | Peek      | O(1) amortized / O(n) worst               | O(1)                        |
            | Empty     | O(1)                                      | O(1)                        |
