1. Launch EC2 Instance

- Choose Amazon Linux
- Open ports:
  - `22` (SSH)
  - `80` (HTTP)
  - `443` (HTTPS)

2. Connect to EC2

```
ssh -i mykey.pem ec2-user@<public-ip>
```

3. Install Node.js

```
sudo yum update -y
curl -fsSL https://rpm.nodesource.com/setup_18.x | sudo bash -
sudo yum install nodejs -y
```

4. Upload & Run App

```
git clone https://github.com/your-repo/node-app.git
cd node-app
npm install
node index.js
```

At this point, the app runs â€” but stops if the terminal closes

5. Use PM2 for Production

```
sudo npm install -g pm2
pm2 start index.js --name aws-node-app
pm2 startup
pm2 save
```

Now:

- App runs in background
- Restarts on crash
- Auto-starts on reboot
